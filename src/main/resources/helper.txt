Syntaxe : java [-options] -jar xxxxx.jar

* où options :
    *    -D<name><value>
        *          port     : le port du serveur FTP, par défaut 2121
        *          dir      : chemin du répertoire à partager, par défaut user.home de java
        *          auth     : nomUtilisateur:motDePasse, par defaut admin:admin
